# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in RapidTools Validation API, please report it to:

**Email:** security@rapidtools.dev

## Disclosure Policy

Full disclosure policy and incident response procedures: https://directory.rapidtools.dev/security

## What to Include

- **Service affected**: Validation API
- **Impact**: What could be compromised or exploited
- **Steps to reproduce**: Clear reproduction steps
- **Environment**: API version, endpoint, request details

## Response Time

We aim to acknowledge reports within 48 hours.

## Scope

**In scope:**
- API endpoints (validation-api.rapidtools.dev)
- Authentication and authorization mechanisms
- Data validation and idempotency handling
- Manifest integrity

**Out of scope:**
- Social engineering attacks
- Physical security
- Third-party services (Cloudflare, email providers)
- Denial of service attacks
